# Complete Linear Regression Automation

基于Python实现的**自动化线性回归建模工具**，封装了从数据预处理、模型训练、超参数调优到结果评估、可视化的全流程，支持普通线性回归、Ridge、Lasso、ElasticNet四种模型，可快速应用于回归类任务，提升建模效率与规范性。

## 📌 项目简介

本项目核心是一个高度封装的`CompleteLinearRegression`类，整合了机器学习建模的全链路流程，无需手动编写繁琐的预处理、调参、评估代码，仅需几行调用即可完成完整的回归建模，适合数据分析、机器学习入门实践及中小型回归任务快速落地。

项目亮点：面向对象封装、多模型支持、自动超参数搜索、全指标评估、中文可视化、代码可读性强，完全符合工业级建模规范。

## 🔧 核心功能

- **数据预处理**：自动数据划分（训练集/测试集）、特征标准化、多项式特征构造，适配不同复杂度的回归场景。
- **多模型支持**：集成4种经典回归模型，可自由切换：
  - 普通线性回归（LinearRegression）
  - 岭回归（Ridge）- 解决多重共线性
  - Lasso回归 - 特征选择与正则化
  - 弹性网络（ElasticNet）- 结合L1和L2正则化
- **自动超参数调优**：基于网格搜索（GridSearchCV）实现超参数自动搜索，无需手动调试，提升模型性能。
- **全面模型评估**：输出训练集/测试集多维度指标（R²、MAE、MSE），支持K折交叉验证，量化模型泛化能力。
- **可视化工具**：自动生成学习曲线（判断过拟合/欠拟合）、特征重要性图（排列重要性），直观分析模型性能。
- **详细报告输出**：自动生成模型训练摘要，包含性能指标、超参数配置、过拟合分析及总体评级，便于快速复盘。
